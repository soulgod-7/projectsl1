Breadboard

🔧 Basic Information

  • Name: Breadboard (also called solderless breadboard)
  • Purpose: Used to build and test electronic circuits without soldering

🧱 Structure

  • Material: Typically plastic body with metal strips inside
  • Main Sections:
    • Terminal Strips (main area): Where you place components
    • Bus Strips (power rails): For power distribution (labeled + and –)

📐 Pin Layout

  • Standard Size:
    • Half-size breadboard: ~400 tie-points
    • Full-size breadboard: ~830 tie-points
  • Rows and Columns:
    • Terminal area: 2 columns of 5 connected holes (labeled A–E and F–J)
    • Power rails: Long columns marked + and – (each side)

⚡ Electrical Connections

  • Rows A–E and F–J: Each set of 5 holes is connected horizontally
  • Power Rails: Columns are connected vertically
  • Central Gap: Used for placing ICs (separates connections for each side)

⚙️ Features

  • Reusability: Components can be inserted and removed many times
  • No soldering needed
  • Supports low-voltage circuits (3.3V or 5V)
  • Compatible with jumper wires and Arduino boards

🛠️ Common Uses

  • Prototyping circuits
  • Educational electronics experiments
  • Testing sensors, modules, and microcontrollers
  • Developing quick hardware demos
